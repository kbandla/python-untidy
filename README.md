untidy - XML Fuzzer
===================

More info at
============

http://untidy.sourceforge.net/

Usage
=====

import untidy

xmlString = '<xml attr1="a"><a>f00!</a></xml>'

xf = untidy.xmlFuzzer()
xf.setRepetitions( [3,30,60] )
iter = xf.fuzz( xmlString )

for i in iter:
     print i
